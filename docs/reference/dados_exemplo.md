# Dados de peneiramento de exemplo

Um conjunto de dados fictício contendo resultados de peneiramento de
solo para demonstrar as funções do pacote.

## Usage

``` r
dados_exemplo
```

## Format

Um data frame com 10 linhas e 3 colunas:

- amostra_id:

  Identificação da amostra

- diametro_mm:

  Diâmetro da peneira em mm

- massa_g:

  Massa de solo retida em gramas
